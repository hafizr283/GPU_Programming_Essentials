## নতুন পাওয়া গেল

### 1. `cudaDeviceSynchronize()` — সবচেয়ে Silent Killer

```cpp
compute_RHS_kernel<<<1, K>>>(u, ...);
cudaDeviceSynchronize(); // ← পুরো GPU block হয়ে যাচ্ছে প্রতি iteration-এ
```

এটা GPU-কে প্রতিটা user/item-এর পর **host পর্যন্ত sync** করাচ্ছে। `cudaStreamSynchronize(stream)` দিয়ে replace করলে শুধু ঐ stream wait করবে, বাকি কাজ চলতে পারবে।

### 2. `d_info` কখনো check হচ্ছে না

```cpp
cusolverDnSpotrf(..., d_info); // Cholesky fail করলেও জানার উপায় নেই
cusolverDnSpotrs(..., d_info); // wrong result নিয়ে loop চলতে থাকে
```

Cholesky fail মানে matrix singular বা near-singular — lambda ঠিক না থাকলে এটা হয়। Benchmark-এর সময় wrong convergence দেখাবে।
