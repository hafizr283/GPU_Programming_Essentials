## 📚 Core Papers (অবশ্যই পড়তে হবে)

**BALS — আপনার base paper**
https://ieeexplore.ieee.org/document/9373912/
Chen et al., "BALS: Blocked Alternating Least Squares for Parallel Sparse Matrix Factorization on GPUs", IEEE TPDS 2021

**cuMF_ALS — Approximate Computing precursor**
https://arxiv.org/pdf/1808.03843
Tan et al., "Matrix Factorization on GPUs with Memory Optimization and Approximate Computing", ICPP 2018

**cuMF_SGD — GPU MF foundation**
https://arxiv.org/pdf/1610.05838
Xie et al., "CuMF_SGD: Fast and Scalable Matrix Factorization", HPDC 2017

**Koren classic — MF basics**
https://datajobs.com/data-science-repo/Recommender-Systems-[Netflix].pdf
Koren, Bell, Volinsky, "Matrix Factorization Techniques for Recommender Systems", IEEE Computer 2009

## 🎯 Mixed Precision — Tile/Block Level (closest related work)

**BDMP — সবচেয়ে কাছাকাছি কাজ**
https://link.springer.com/article/10.1007/s11227-024-05949-6
"Block-wise Dynamic Mixed-Precision for SpMV on GPUs", J. Supercomputing 2024

**PaRSEC tile-centric mixed precision**
https://arxiv.org/html/2508.14848
"Leveraging Hardware-Aware Computation in Mixed-Precision Matrix Multiply: A Tile-Centric Approach", 2025

**tSparse — Tensor Core spGEMM**
https://arxiv.org/pdf/2009.14600
"Accelerating Sparse Matrix-Matrix Multiplication with GPU Tensor Cores", 2020

**Mixed-precision iterative refinement (foundational)**
https://www.ncbi.nlm.nih.gov/pmc/articles/PMC7735315/
Haidar et al., "Mixed-precision iterative refinement using tensor cores on GPUs", 2020
